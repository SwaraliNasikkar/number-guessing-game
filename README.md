# 🎯 Number Guessing Game CLI 

Hey there! I'm super happy you're here ❤️  
This is a simple, beginner-friendly **Number Guessing Game** built using Python 🐍  
If you enjoy fun logic games or are just starting your coding journey — this one’s for you! ✨

---

## 🧩 Features

- 🔢 Random number generation (1–100)
- 🎯 User guesses the number
- 📉 Hints:
  - Too high → go lower!
  - Too low → go higher!
- 🎉 Celebration message when you win
- ⏳ Keeps running until correct guess

---

## 💻 Technologies Used

- Python (loops, conditionals, input/output)
- Terminal (CLI)
- Logic 🧠 + Fun 💖
